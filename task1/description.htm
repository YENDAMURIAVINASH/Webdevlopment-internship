🎲About Dice Roller Game

The Dice Roller Game is a simple and interactive web application designed to simulate the rolling of a six-sided dice.
Built using HTML, CSS, and JavaScript, the game offers an engaging way for users to test their luck and basic number intuition.

🧩 Features:
  • 🎯 User Input: Users enter their guess (a number between 1 and 6) for the dice roll.

  • 🎲 Dice Simulation: Upon clicking the “Roll Dice” button, a random number is generated and the corresponding dice image is displayed.

  • 🏆 Result Feedback: The game compares the user’s guess with the rolled number and displays a result:
    “🎉 You Win!” with celebration emoji if guessed correctly"
    “😞 You Lose.” if the guess was incorrect"

  • 📷 Dice Image: Shows a real-time dice image update based on the roll result.

  • > 🔁 Play Again Button: Lets users reset and try again.

  • 🏠 Home Button: Takes users back to the home page (if implemented).

  • 📱 Responsive Design: Clean and centered layout that adapts to different screen sizes.

  • 🖼️ Stylish UI: Includes a header and footer with custom colors, spacing, and shadows for a polished look.
👨‍💻 Technologies Used:
  • HTML: Structure of the webpage.

  • CSS: Styling and layout (header, footer, buttons, container).

  • JavaScript: Dice roll logic, DOM manipulation, user interaction.
